Sentiment Analysis of Social Media Conversations

Decoding human emotions from social media text using Natural Language Processing (NLP)

📌 Project Overview
This project builds an end-to-end NLP pipeline to detect and classify emotions and sentiments from social media conversations. It helps understand public opinion, user behavior, and emotional trends from raw text data.
Completed as part of the IBM Data Science with Python Internship (July 2025).

🎯 Objectives

Clean and preprocess raw social media text data
Build a sentiment classification pipeline (Positive / Negative / Neutral)
Visualize sentiment trends and emotional patterns
Apply real-world NLP techniques to unstructured data

🔍 Key NLP Steps

Text Preprocessing — Lowercasing, removing stopwords, punctuation cleaning
Tokenization — Splitting text into meaningful tokens
Sentiment Scoring — Using TextBlob polarity scores
Emotion Classification — Positive / Negative / Neutral labeling
Visualization — Trend graphs and word frequency plots
